# Data Structures and Algorithms (DSA) Solutions

Welcome to my DSA repository! This collection contains my solutions to various algorithmic problems and data structure challenges, mostly sourced from LeetCode, GeeksforGeeks, and other coding platforms. The solutions are primarily written in **JavaScript**.

## 📂 Repository Structure

The problems are categorized based on the core algorithmic pattern or data structure used to solve them. Here is a breakdown of the topics covered:

- **2D Array**: Matrix traversals, transformations, and coordinate-based problems.
- **Binary Search**: Efficient searching algorithms in sorted arrays and search spaces.
- **Interview Javascript**: Common JavaScript specific interview questions and polyfills.
- **Kadane / Sliding Window**: Optimal contiguous subarray problems (Maximum sum, etc.).
- **Merge Intervals**: Managing overlapping intervals and scheduling problems.
- **Prefix Sum**: Precomputing cumulative sums for fast range queries.
- **Sliding Window**: Optimization problems on contiguous subarrays or substrings.
- **Slow and Fast Pointers**: Cycle detection and finding elements in linked lists or arrays.
- **Two Pointers**: Efficiently iterating through collections from both ends or at different speeds.
- **XOR**: Bit manipulation techniques specifically leveraging XOR properties.
- **Misc**: Assorted problems including sorting algorithms, array rotations, etc.

## 🚀 How to Use

Each folder contains JavaScript files named after the specific problem. 
You can run these files using Node.js in your terminal:

```bash
node "Folder Name/File Name.js"
```
